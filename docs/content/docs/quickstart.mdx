Get from zero to a production-ready API in minutes. YantrJS supports **Express**, **Hono**, and **Fastify**.

## 1. Installation

Run YantrJS directly using `npx`:

```bash
npx yantr-js init
```

Or install globally:

```bash
npm install -g yantr-js
```

## 2. Initialize your Project

Run the `init` command in your project directory:

```bash
yantr init
```

YantrJS will:
1. **Detect or create** your `package.json`
2. **Ask which framework** you want (Express, Hono, or Fastify)
3. **Create** `yantr.json` configuration
4. **Install** framework-specific base templates

> **Tip:** YantrJS can start a brand new project! Run `yantr init` in an empty directory and it will create everything for you.

## 3. Add a Database

Set up your database with ORM of choice:

```bash
yantr add database
```

Select from:
- **PostgreSQL** with Prisma or Drizzle
- **MongoDB** with Mongoose

## 4. Add Components

Add production-ready features:

```bash
yantr add auth     # JWT authentication
yantr add logger   # Pino structured logging
yantr add security # Rate limiting + Helmet headers
```

## 5. Generate Routes

Scaffold a new API resource:

```bash
yantr generate route users
```

Creates:
- `routes/users.routes.ts`
- `controllers/users.controller.ts`
- `services/users.service.ts`

## 6. Wire it up

### Express

```typescript
import express from 'express';
import { errorHandler } from './lib/yantr/error-handler';
import { httpLogger } from './lib/yantr/logger/http-logger';
import usersRoutes from './routes/users.routes';

const app = express();
app.use(express.json());
app.use(httpLogger);
app.use('/api/users', usersRoutes);
app.use(errorHandler); // Must be last

app.listen(3000);
```

### Hono

```typescript
import { Hono } from 'hono';
import { onError } from './lib/yantr/error-handler';
import usersRoutes from './routes/users.routes';

const app = new Hono();
app.onError(onError);
app.route('/api/users', usersRoutes);

export default app;
```

### Fastify

```typescript
import Fastify from 'fastify';
import { errorHandler } from './lib/yantr/error-handler';

const fastify = Fastify();
fastify.setErrorHandler(errorHandler);

fastify.listen({ port: 3000 });
```
